## How to run locally
Run these commands on CMD or terminal
> git clone https://github.com/fikrisyahid/teks-gaya-converter.git

> cd teks-gaya-converter

if you use npm

> npm install

> npm run dev

or if you use Yarn

> yarn

> yarn dev

Open `http://localhost:3000` in your browser
